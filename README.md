## ⚙️ Installation & Setup

Most projects are designed for Linux environments. Ensure you have `gcc`, `g++`, `rustc`, and `python3` installed.

```bash
# Example: Running the Rule 110 implementation
cd rule-110-rust
cargo run

# Example: Compiling the Multi-threaded Server
cd network-app
gcc -pthread main.c -o server
./server
```
