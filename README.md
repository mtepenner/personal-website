# Matthew Penner | Official Webpage

## Description
A dynamic, cyberpunk-themed personal portfolio website built to showcase my software engineering projects, technical skills, and professional experience. It serves as a central hub for my resume, academic background at Portland State University, and a collection of custom web applications.

## Table of Contents
- [Features](#features)
- [Technologies Used](#technologies-used)
- [Installation](#installation)
- [Usage](#usage)

## Features 🚀
* **Modern, Cyberpunk UI:** Custom CSS properties featuring neon glows, grid backgrounds, and tech-inspired typography (`Orbitron`, `Share Tech Mono`).
* **Interactive Navigation:** A sticky navigation bar with dropdown menus routing to various hosted web apps (Games, News, Calculator, ISS Tracker, etc.).
* **Comprehensive Professional Profile:** Detailed sections highlighting my "About Me", "Education", and "Experience" (including contract work at Intel Corporation).
* **Skill Showcase:** A visually distinct tag-based layout for my technical stack, including C/C++, Python, Rust, React, and Node.js.
* **Direct Asset Integration:** Easy access links to my GitHub profile, LinkedIn, and downloadable PDF resume.

## Technologies Used 🛠️
* **HTML5** for semantic structuring.
* **CSS3** for responsive design, flexbox/grid layouts, and advanced visual effects (gradients, text-shadows, backdrop filters).
* **Vanilla JavaScript** for smooth scrolling and interactive DOM manipulation.
* **Google Fonts** integration for stylized text rendering.

## Installation
As a static website, no complex build steps or dependencies are required to run this project locally.

1. Clone the repository to your local machine:
   ```bash
   git clone [https://github.com/mtepenner/personal-website.git](https://github.com/mtepenner/personal-website.git)
    ````

2.  Navigate into the project directory:
    ```bash
    cd personal-website
    ```
3.  Open `index.html` in any modern web browser to view the site.

## Usage

  * **Navigation:** Click the links in the header navigation to smoothly scroll to the `About`, `Skills`, or `Contact` sections.
  * **Web Apps:** Hover over the "Web Apps" tab to reveal the dropdown menu and access sub-projects like the ISS Tracker or Oil Conversions tools.
  * **Details:** Click on the "Read More..." summary tags within the About and Education cards to expand and view deeper details about my background and senior thesis.
